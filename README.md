# ArchitectureBattle

Minecraft JE Server & Client Side Fabric Mod

建築練習等に役立つと嬉しいです⛏

## Usage/使い方

### theme command / テーマコマンド
```
/theme <> <> ...
```

### timer command / タイマーコマンド
```
/timer <minutes>
```
pause / 一時停止
```
/timer stop
```
resume / 再開
```
/timer start
```
